# HWK2 Code - Shreesh Karjagi

## Files
- prob2_inference.py  : Problem 2: Variable Elimination inference code
- prob5_partition.py  : Problem 5: Partition function computation (butterfly method)
- prob6_bearbull.py   : Problem 6: Bear/Bull HMM forward algorithm

## Running the code

### Problem 2
```
python3 prob2_inference.py
```
Requires: numpy, joint.dat, dataset.dat using the file path at the bottom

### Problem 5
```
python3 prob5_partition.py
```
Requires: numpy. Computes log(Z) for n=20 lattice MRF

### Problem 6
```
python3 prob6_bearbull.py
```
Requires: numpy, scipy (for loading .mat file), BearBullproblem.mat. using the file path at the top
